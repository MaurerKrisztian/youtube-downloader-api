#  Youtube downloader api

Blog post: https://dev.to/maurerkrisztian/youtube-downloader-cli-wrapper-2h61

Setup:

create a .env file:
```
SELF_URL= # optional, default localhost
PORT= # optional
LOG_LEVEL= # optional
USE_BIN= # optional, use local bin or env, default bin
```
